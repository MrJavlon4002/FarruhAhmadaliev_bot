Технология преобразования речи в текст (STT):
Он транскрибирует аудио на узбекском, русском и английском языках с высокой точностью.
 - Может конвертировать узбекские, русские и английские аудиофайлы в текстовый формат.
 - Конвертирует аудиофайл в текстовый формат с точностью 90%.
 - Эффективно обрабатывает голос и обеспечивает точную транскрипцию даже в шумной обстановке.
 - Он может различать речь нескольких говорящих с помощью функции диаризации.
 - Функция 5 Voice ID позволяет распознавать голоса и идентифицировать людей.
 - Благодаря нашим усовершенствованным моделям он также может переводить узбекские диалекты на художественный язык.

Технология преобразования текста в речь (TTS):
Преобразует тексты на узбекском, русском и английском языках в естественное звучание.
 - Лучшая модель для преобразования узбекского текста в речь.
 - Возможность выбора пола голоса — доступны женские и мужские голоса.
 - Человек воспроизводит естественный звук со 100% точностью.
 - Также может озвучивать тексты на русском и английском языках.
 - Реагирует на знаки препинания, добавляя паузы, акценты и другие типичные речевые тенденции.
 - Самая быстрая модель преобразования текста в речь на узбекском языке.

Голосовой идентификатор:
Современная технология голосовой биометрии, которая анализирует уникальные характеристики голоса пользователя и использует их для его идентификации.
 - Идентификация и аутентификация. Точная проверка личности на основе уникальных акустических характеристик голоса каждого человека
 - Повысить безопасность. Повышение безопасности процесса аутентификации за счет снижения необходимости использования паролей или ПИН-кодов.
 - Удобство и скорость. Пользователь может войти в систему или воспользоваться услугами с помощью голоса, что экономит время.
 - Постоянное обучение. Способность обучаться и адаптироваться к изменениям голоса с помощью искусственного интеллекта.
 - Поддержка нескольких платформ. Возможность использования системы для телефонных звонков, чат-ботов и других голосовых сервисов.
 - Защищенные биометрические данные. Голосовые данные пользователя хранятся в зашифрованном виде и используются только для целей аутентификации.

 API для разработчиков:
Текст в речь:
 - TTS API позволяет преобразовывать текст в речь. Вы можете использовать его для создания голосовых сообщений, голосовых ботов и других голосовых приложений.
 - Audio API предоставляет речевую конечную точку на основе нашей модели TTS (текст-в-речь). Он поставляется с 6 встроенными голосами и может использоваться для следующих целей:
 1. Озвучивание письменного сообщения в блоге.
 2. Создайте закадровый голос на нескольких языках.
 - Для преобразования речи в текст необходимо отправить POST-запрос на /api/v1/tts/post с данными JSON:
 1. стенограмма - текст. Преобразуйте текст в речь.
 2. язык - текст. Значения: «uz»/«en»/«ru». Мы поддерживаем узбекский, английский и русский языки.
 3. модель - текст. Значения — «цветочный»/«глобальный». У нас есть модели для девочки (Гульноза) и мальчика (Джахонгир).
 - Код:
 ```
 curl --request POST
 --url https://back.aisha.group/api/v1/tts/post/
 --header 'x-api-key: [API_KEY]'
 --header 'Тип содержимого: multipart/form-data'
 --header 'X-каналы: стерео'
 --header 'X-Качество: 64k'
 --header 'X-Скорость: 16000'
 --header 'X-Формат: mp3'
 --form 'transcript=Ассаляму алейкум'
 --form 'язык=uz'
 --form 'run_diarization=false'
 --form 'модель=цветок'
 ```
Преобразование речи в текст:
 - API STT позволяет преобразовывать речь в текст.
 - API аудио предоставляет две конечные точки для преобразования речи в текст — транскрипцию и перевод — на основе нашей современной модели с открытым исходным кодом Large v2 Whisper. Их можно использовать в следующих целях:
 1. Транскрибируйте аудиозапись на узбекский, русский и английский языки в зависимости от языка оригинала.
 - Для преобразования речи в текст необходимо отправить запрос POST /api/v1/stt/post с данными формы:
 1. аудио - файл. Преобразование речи в текст.
 2. язык - текст. Значения: «uz»/«en»/«ru». Мы поддерживаем узбекский, английский и русский языки.
 - Код:
 ```
 curl --request POST
 --url https://back.aisha.group/api/v1/stt/post/
 --header 'x-api-key: [API_KEY]'
 --header 'Тип содержимого: multipart/form-data'
 --form 'audio=/путь/к/файлу'
 --form 'has_offsets=false'
 --form 'has_diarization=false'
 --form 'язык=uz'
 ```
